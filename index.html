Premios y UIF
El sistema no encuentra el ticket al cargar un premio+

Ingresar solo los dígitos del número de secuencia, sin espacios.

Autogestión
Error en CUIT/CUIL al cargar el beneficiario+

Si el DNI tiene 7 dígitos, agregar un 0 antes. Ej: 20061234561.

Autogestión
Error "Control 15b — debe ingresar la característica"+

Teléfono completado sin código de área. Agregar: 342 para Santa Fe, 341 para Rosario.

Autogestión
El sistema no acepta el CBU del ganador+

Se ingresó un CVU (Mercado Pago, Ualá, Prex). Solicitar el CBU bancario. Brubank y Naranja X son aceptados.
